# Think Decision — Frontend

React + TypeScript + Vite + Tailwind CSS frontend untuk platform AHP multi-pakar.

## Setup

```bash
cd think-decision-frontend
npm install
cp .env.example .env
npm run dev
```

Akses di: http://localhost:5173

## Build untuk Production

```bash
npm run build
npm run preview
```

## API Integration

Ubah `VITE_API_URL` di `.env` sesuai backend URL Anda (default: http://localhost:8000/api)
